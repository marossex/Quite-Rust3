# Quite Rust - Официальный сайт сервера

Красивый сайт для сервера Rust Quite Rust с информацией, статистикой и быстрым доступом.

## 🚀 Быстрый старт

### Требования:
- Node.js 18+ (скачать с https://nodejs.org/)

### Установка и запуск:

1. **Открой PowerShell в папке проекта**
   ```powershell
   cd C:\Users\Рома\quite-rust-site
   ```

2. **Установи зависимости**
   ```powershell
   npm install
   ```

3. **Запусти сайт в режиме разработки**
   ```powershell
   npm run dev
   ```

4. **Открой в браузере**
   - Перейди на http://localhost:3000

## 📄 Структура проекта

## 🔧 Редактирование

### Изменить информацию о сервере:
Открой `app/server/page.js` и найди строку:
```javascript
<p>Добро пожаловать на сервер...</p>
```
Замени текст на свой.

### Добавить статистику онлайна с сервера:
В `app/server/page.js` раскомментируй код в `useEffect`:
```javascript
// const fetchOnline = async () => {
//   const res = await fetch('/api/server/status')
//   const data = await res.json()
//   setOnlinePlayers(data.online)
// }
// fetchOnline()
```

И создай API endpoint в `app/api/server/status/route.js` для запроса данных с сервера.

## 🌐 Развёртывание

### На Vercel (бесплатно):
1. Загрузи проект на GitHub
2. Зайди на https://vercel.com
3. Импортируй репозиторий
4. Нажми Deploy

### На своём хостинге:
```powershell
npm run build
npm run start
```

## 📞 Поддержка

Если что-то не работает, проверь:
1. Node.js установлен (`node -v`)
2. Все зависимости установлены (`npm install`)
3. Порт 3000 свободен
